# Screenshots

Place your 9:16 Android app screenshots here with these exact filenames:

| Filename                     | Screen                        | Used in                        |
|------------------------------|-------------------------------|--------------------------------|
| `screen-dashboard.png`       | Home Dashboard                | Hero (front phone) + Walkthrough §1 |
| `screen-tenants.png`         | Tenants list / tenant detail  | Hero (back phone)              |
| `screen-payments.png`        | Payments / billing ledger     | Walkthrough §2                 |
| `screen-documents.png`       | Document cabinet              | Walkthrough §3                 |
| `screen-settings.png`        | Settings / property config    | Walkthrough §4                 |

## Requirements
- Format: PNG (JPEG also works, just rename extension in index.html if needed)
- Aspect ratio: 9:16 portrait (e.g. 1080×1920, 1440×2560, 720×1280)
- No landscape screenshots
- Capture at whatever native resolution your device uses — the site scales them automatically

## Notes
- All 5 are optional — a neat placeholder box shows if a file is missing
- The hero looks best when `screen-dashboard.png` and `screen-tenants.png` are both present
- Screenshots can be captured from Android Studio Emulator or directly from your device
